Projet de première année d’études supérieures. Réalisé en binôme, nous avons développé un site web complet front-end et back-end pour suivre la Ligue 1, en utilisant les langages suivants : HTML, CSS, PHP, JavaScript et MySQL.

Fonctionnalités :
 • Système d’inscription et de connexion des utilisateurs ;
 • Recherche et suivi des clubs et des rencontres ;
 • Suivi en temps réel du classement et mise à jour automatique ;
 • Système de commentaires sur les rencontres ;
 • Section FAQ pour les questions fréquentes.

Lien vers le site : Ligue1.42web.io￼

Technologies : HTML, CSS, PHP, JavaScript, MySQL
Compétences développées : Développement front-end et back-end, gestion de base de données, systèmes d’authentification, interactivité utilisateur.
